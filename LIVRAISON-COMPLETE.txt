═══════════════════════════════════════════════════════════════════════════
                       🚀 DÉMARRAGE RAPIDE (5 ÉTAPES)
═══════════════════════════════════════════════════════════════════════════

1. INSTALLER LES DÉPENDANCES
   > npm install

2. CONFIGURER LES VARIABLES D'ENVIRONNEMENT
   • Copier .env.example vers .env
   • Remplir VITE_APPWRITE_PROJECT_ID (requis)
   • Remplir VITE_GEMINI_API_KEY (optionnel)

3. LANCER L'APPLICATION
   > npm run dev
   → Accéder à http://localhost:5173

4. CONFIGURER APPWRITE (Pour persistance)
   • Créer compte sur cloud.appwrite.io
   • Créer projet "Ecosyst5a"
   • Créer 22 collections (voir docs/02-CAHIER-DES-CHARGES.md)
   • Copier Project ID dans .env

5. BUILD PRODUCTION
   > npm run build
   → Déployer dossier /dist sur Vercel/Netlify
